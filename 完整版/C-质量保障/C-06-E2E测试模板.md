## 测试结构

### 通用钩子

```
全局前置钩子:
  - 初始化测试环境
  - 配置测试数据
  - 启动被测服务

全局后置钩子:
  - 清理测试数据
  - 关闭被测服务
  - 生成测试报告

用例前置钩子:
  - 准备用例数据
  - 导航到目标页面

用例后置钩子:
  - 截图（失败时）
  - 清理用例数据
```

### 用例编写规范

```
用例结构:
  1. 用例描述: 清晰说明测试目标
  2. 前置条件: 列出必要的前置状态
  3. 测试步骤: 按顺序列出操作
  4. 预期结果: 明确的断言条件
  5. 清理操作: 恢复环境状态
```
