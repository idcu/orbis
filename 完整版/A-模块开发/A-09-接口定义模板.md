以下是一个完整的接口定义文档示例：

```markdown
# string-utils 接口定义

## 模块概述

通用字符串处理工具集，提供常用的字符串操作函数。

**模块类别**：工具库

**稳定性**：稳定

---

## 接口列表

| 接口 | 类型 | 稳定性 | 说明 |
|------|------|--------|------|
| reverse | 函数 | 稳定 | 反转字符串 |
| capitalize | 函数 | 稳定 | 首字母大写 |
| truncate | 函数 | 稳定 | 截断字符串 |
| StringCase | 类型 | 稳定 | 字符串大小写枚举 |

---

## 详细接口定义

### reverse

\`\`\`
reverse(input: string) -> string
\`\`\`

**描述**：将输入字符串反转并返回结果。

**所属分组**：变换操作

**稳定性**：稳定

**自版本**：0.1.0

#### 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| input | string | 是 | - | 待反转的字符串 |

#### 返回值

`string` — 反转后的字符串

#### 使用示例

**基本用法**：

\`\`\`
reverse("hello")  // 返回 "olleh"
\`\`\`

**高级用法**：

\`\`\`
reverse("Hello, World!")  // 返回 "!dlroW ,olleH"
\`\`\`

#### 错误处理

| 错误条件 | 错误类型 | 错误码 | 说明 |
|----------|----------|--------|------|
| 输入非字符串 | TypeError | INVALID_INPUT | 参数必须为字符串类型 |

#### 关联接口

| 接口 | 关系 | 说明 |
|------|------|------|
| capitalize | 互补 | 字符串格式化相关 |

---

### StringCase

\`\`\`
type StringCase = "lower" | "upper" | "camel" | "snake" | "kebab"
\`\`\`

**描述**：字符串大小写格式枚举，用于指定字符串的格式化方式。

**字段说明**：

| 值 | 说明 |
|----|------|
| lower | 全小写 |
| upper | 全大写 |
| camel | 驼峰命名 |
| snake | 蛇形命名（下划线分隔） |
| kebab | 短横线命名 |
```
