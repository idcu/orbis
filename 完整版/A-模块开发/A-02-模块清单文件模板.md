# A-F02 模块清单文件模板（完整版）

> **模板 ID**: A-F02
> **模板名称**: 模块清单文件模板（完整版）
> **版本**: 1.0.0
> **适用范围**: 模块/包的清单文件完整配置，覆盖多种模块类型

---

## 模板说明

模块清单文件是每个模块的核心元数据文件，声明了模块的基本信息、入口、依赖、脚本命令和发布配置。

不同生态系统使用不同的清单文件格式，但核心概念相通。本模板使用通用的 JSON-like 格式，
可映射到任意生态系统：

| 生态系统 | 清单文件名 | 格式 |
|----------|-----------|------|
| 生态 A | `package.json` | JSON |
| 生态 B | `Cargo.toml` | TOML |
| 生态 C | `go.mod` | 自定义 |
| 生态 D | `build.gradle` / `pom.xml` | Groovy / XML |

---

## 通用字段参考

| 字段 | 必填 | 说明 |
|------|------|------|
| `name` | 是 | 模块名称 |
| `version` | 是 | 版本号（语义化版本） |
| `description` | 是 | 模块描述 |
| `main` | 是 | 主入口文件 |
| `module` | 否 | ESM 入口文件 |
| `types` | 否 | 类型声明入口 |
| `exports` | 否 | 条件导出映射 |
| `files` | 否 | 发布白名单 |
| `bin` | 否 | 可执行文件入口 |
| `scripts` | 是 | 脚本命令 |
| `dependencies` | 否 | 运行时依赖 |
| `devDependencies` | 否 | 开发依赖 |
| `peerDependencies` | 否 | 对等依赖（宿主环境提供） |
| `sideEffects` | 否 | 副作用标记 |
| `license` | 是 | 许可证 |
| `repository` | 否 | 仓库地址 |
| `keywords` | 否 | 搜索关键词 |
| `author` | 否 | 作者信息 |
| `contributors` | 否 | 贡献者列表 |
| `engines` | 否 | 运行环境版本要求 |

## 变体一：零依赖模块

适用于不依赖任何外部运行时库的独立工具模块。

```json
{
  "name": "@{{SCOPE}}/{{MODULE_NAME}}",
  "version": "0.1.0",
  "description": "{{DESCRIPTION}}",
  "main": "src/index",
  "module": "src/index",
  "types": "src/index.d",
  "exports": {
    ".": {
      "types": "src/index.d",
      "import": "src/index.mjs",
      "require": "src/index.cjs"
    }
  },
  "files": [
    "src",
    "README.md",
    "CHANGELOG.md",
    "LICENSE"
  ],
  "sideEffects": false,
  "scripts": {
    "build": "{{BUILD_COMMAND}}",
    "test": "{{TEST_COMMAND}}",
    "lint": "{{LINT_COMMAND}}",
    "typecheck": "{{TYPECHECK_COMMAND}}"
  },
  "dependencies": {},
  "devDependencies": {
    "{{BUILD_TOOL}}": "{{BUILD_TOOL_VERSION}}",
    "{{TEST_FRAMEWORK}}": "{{TEST_FRAMEWORK_VERSION}}",
    "{{LINTER}}": "{{LINTER_VERSION}}",
    "{{TYPECHECK_TOOL}}": "{{TYPECHECK_TOOL_VERSION}}"
  },
  "license": "{{LICENSE}}",
  "repository": {
    "type": "{{REPO_TYPE}}",
    "url": "{{REPO_URL}}"
  },
  "keywords": ["{{KEYWORD_1}}", "{{KEYWORD_2}}"],
  "author": "{{AUTHOR}}"
}
```

## 变体二：有依赖模块

适用于依赖仓库内其他模块或外部库的模块。

```json
{
  "name": "@{{SCOPE}}/{{MODULE_NAME}}",
  "version": "0.1.0",
  "description": "{{DESCRIPTION}}",
  "main": "src/index",
  "module": "src/index",
  "types": "src/index.d",
  "exports": {
    ".": {
      "types": "src/index.d",
      "import": "src/index.mjs",
      "require": "src/index.cjs"
    }
  },
  "files": [
    "src",
    "README.md",
    "CHANGELOG.md",
    "LICENSE"
  ],
  "sideEffects": false,
  "scripts": {
    "build": "{{BUILD_COMMAND}}",
    "test": "{{TEST_COMMAND}}",
    "lint": "{{LINT_COMMAND}}",
    "typecheck": "{{TYPECHECK_COMMAND}}"
  },
  "dependencies": {
    "@{{SCOPE}}/{{INTERNAL_DEP}}": "{{WORKSPACE_PROTOCOL}}:{{WORKSPACE_PATH}}",
    "{{EXTERNAL_DEP}}": "{{EXTERNAL_DEP_VERSION}}"
  },
  "devDependencies": {
    "{{BUILD_TOOL}}": "{{BUILD_TOOL_VERSION}}",
    "{{TEST_FRAMEWORK}}": "{{TEST_FRAMEWORK_VERSION}}",
    "{{LINTER}}": "{{LINTER_VERSION}}",
    "{{TYPECHECK_TOOL}}": "{{TYPECHECK_TOOL_VERSION}}"
  },
  "peerDependencies": {
    "{{PEER_DEP}}": "{{PEER_DEP_VERSION_RANGE}}"
  },
  "license": "{{LICENSE}}",
  "repository": {
    "type": "{{REPO_TYPE}}",
    "url": "{{REPO_URL}}"
  },
  "keywords": ["{{KEYWORD_1}}", "{{KEYWORD_2}}"],
  "author": "{{AUTHOR}}"
}
```

**依赖说明**：

| 依赖类型 | 字段 | 说明 |
|----------|------|------|
| 内部依赖 | `dependencies` | 使用工作区协议引用仓库内其他模块 |
| 外部依赖 | `dependencies` | 使用版本范围引用外部库 |
| 对等依赖 | `peerDependencies` | 由宿主环境/使用者提供的依赖 |

## 变体三：CLI 工具模块

适用于提供命令行工具的模块。

```json
{
  "name": "@{{SCOPE}}/{{MODULE_NAME}}",
  "version": "0.1.0",
  "description": "{{DESCRIPTION}}",
  "main": "src/index",
  "bin": {
    "{{COMMAND_NAME}}": "./src/cli-entry"
  },
  "files": [
    "src",
    "README.md",
    "CHANGELOG.md",
    "LICENSE"
  },
  "scripts": {
    "build": "{{BUILD_COMMAND}}",
    "test": "{{TEST_COMMAND}}",
    "lint": "{{LINT_COMMAND}}"
  },
  "dependencies": {
    "{{CLI_FRAMEWORK}}": "{{CLI_FRAMEWORK_VERSION}}"
  },
  "devDependencies": {
    "{{BUILD_TOOL}}": "{{BUILD_TOOL_VERSION}}",
    "{{TEST_FRAMEWORK}}": "{{TEST_FRAMEWORK_VERSION}}",
    "{{LINTER}}": "{{LINTER_VERSION}}"
  },
  "license": "{{LICENSE}}",
  "repository": {
    "type": "{{REPO_TYPE}}",
    "url": "{{REPO_URL}}"
  },
  "keywords": ["{{KEYWORD_1}}", "cli", "command-line"],
  "author": "{{AUTHOR}}",
  "engines": {
    "{{RUNTIME}}": ">= {{MIN_RUNTIME_VERSION}}"
  }
}
```

**CLI 特有字段说明**：

| 字段 | 说明 |
|------|------|
| `bin` | 命令名到可执行文件路径的映射 |
| `engines` | 声明最低运行时版本要求 |

## 变体四：聚合模块（Barrel / Re-export）

适用于将多个子模块统一导出的聚合模块，本身不包含实现代码。

```json
{
  "name": "@{{SCOPE}}/{{MODULE_NAME}}",
  "version": "0.1.0",
  "description": "{{DESCRIPTION}} — 聚合导出模块",
  "main": "src/index",
  "module": "src/index",
  "types": "src/index.d",
  "exports": {
    ".": {
      "types": "src/index.d",
      "import": "src/index.mjs",
      "require": "src/index.cjs"
    }
  },
  "files": [
    "src",
    "README.md",
    "CHANGELOG.md",
    "LICENSE"
  ],
  "sideEffects": false,
  "scripts": {
    "build": "{{BUILD_COMMAND}}",
    "lint": "{{LINT_COMMAND}}"
  },
  "dependencies": {
    "@{{SCOPE}}/{{SUB_MODULE_A}}": "{{WORKSPACE_PROTOCOL}}:{{SUB_MODULE_A_PATH}}",
    "@{{SCOPE}}/{{SUB_MODULE_B}}": "{{WORKSPACE_PROTOCOL}}:{{SUB_MODULE_B_PATH}}",
    "@{{SCOPE}}/{{SUB_MODULE_C}}": "{{WORKSPACE_PROTOCOL}}:{{SUB_MODULE_C_PATH}}"
  },
  "devDependencies": {
    "{{BUILD_TOOL}}": "{{BUILD_TOOL_VERSION}}",
    "{{LINTER}}": "{{LINTER_VERSION}}"
  },
  "license": "{{LICENSE}}",
  "repository": {
    "type": "{{REPO_TYPE}}",
    "url": "{{REPO_URL}}"
  },
  "keywords": ["{{KEYWORD_1}}", "{{KEYWORD_2}}"],
  "author": "{{AUTHOR}}"
}
```

**聚合模块特点**：

- 源码仅包含 re-export 语句，无业务逻辑
- 不需要测试文件（无自身逻辑可测）
- 依赖列表仅包含被聚合的子模块
- `sideEffects` 设为 `false` 以支持 tree-shaking
