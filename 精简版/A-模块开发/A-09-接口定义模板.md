# A-S09 接口定义模板（精简版）

> **模板编号**: A-S09
> **模板名称**: 接口定义模板
> **版本**: v1.0
> **分类**: A-模块开发
> **层级**: 精简版

---

## 模板说明

接口定义文件描述模块对外暴露的所有公开接口，包括函数签名、参数类型和返回值。
以下为精简版模板，适用于快速定义模块 API。

---

## 模板内容

```markdown
# {{MODULE_NAME}} 接口定义

## 概述

{{MODULE_DESCRIPTION}}

## 函数签名

### {{FUNCTION_NAME}}

\`\`\`
{{FUNCTION_NAME}}({{PARAM_NAME}}: {{PARAM_TYPE}}) -> {{RETURN_TYPE}}
\`\`\`

**描述**：{{FUNCTION_DESCRIPTION}}

**参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| {{PARAM_NAME}} | {{PARAM_TYPE}} | 是 | {{PARAM_DESCRIPTION}} |

**返回值**：{{RETURN_TYPE}} — {{RETURN_DESCRIPTION}}

**示例**：

\`\`\`
{{USAGE_EXAMPLE}}
\`\`\`
```

---

## 使用示例

```markdown
# string-utils 接口定义

## 概述

通用字符串处理工具集，提供常用的字符串操作函数。

## 函数签名

### reverse

\`\`\`
reverse(input: string) -> string
\`\`\`

**描述**：将输入字符串反转并返回结果。

**参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| input | string | 是 | 待反转的字符串 |

**返回值**：string — 反转后的字符串

**示例**：

\`\`\`
reverse("hello")  // 返回 "olleh"
\`\`\`
```
