# B-S06 共享脚本模板（精简版）

> **模板 ID**: B-S06
> **模板名称**: 共享脚本模板（精简版）
> **版本**: 1.0.0
> **适用场景**: 基础构建脚本，一键构建工作区中所有包

---

## 概述

本模板提供一个最简构建脚本，按依赖顺序构建工作区中的所有包。

---

## 脚本内容

```bash
#!/bin/sh
# {{项目名称}} - 全量构建脚本
# 文件路径: scripts/build-all.sh

set -e

echo "========================================="
echo "  {{项目名称}} - 全量构建"
echo "========================================="

# --- 检测包管理器 ---
if [ -f "{{包管理器锁文件}}" ]; then
  PKG_MANAGER="{{包管理器命令}}"
elif [ -f "{{备选锁文件}}" ]; then
  PKG_MANAGER="{{备选包管理器命令}}"
else
  echo "错误: 未找到包管理器锁文件"
  exit 1
fi

echo "使用包管理器: $PKG_MANAGER"

# --- 安装依赖 ---
echo ""
echo ">>> 安装依赖..."
$PKG_MANAGER {{安装命令}}

# --- 构建所有包 ---
echo ""
echo ">>> 构建所有包..."
$PKG_MANAGER {{构建命令}} --filter "{{包过滤模式}}"

echo ""
echo "========================================="
echo "  构建完成"
echo "========================================="
```

---

## 占位符说明

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `{{项目名称}}` | 项目名称 | `my-project` |
| `{{包管理器锁文件}}` | 主包管理器的锁文件名 | `pnpm-lock.yaml` |
| `{{备选锁文件}}` | 备选包管理器的锁文件名 | `yarn.lock` |
| `{{包管理器命令}}` | 包管理器可执行命令 | `pnpm` |
| `{{备选包管理器命令}}` | 备选包管理器命令 | `yarn` |
| `{{安装命令}}` | 安装依赖的子命令 | `install` |
| `{{构建命令}}` | 构建的子命令 | `run build` |
| `{{包过滤模式}}` | 选择要构建的包 | `...`（所有包） |

---

## 使用方式

```bash
# 赋予执行权限
chmod +x scripts/build-all.sh

# 运行构建
./scripts/build-all.sh
```

---

## 适配指南

- 根据实际使用的包管理器调整锁文件检测和命令。
- 如需更完整的脚本集（依赖检查、版本同步、测试等），请参考完整版模板（B-F06）。

---

## 注意事项

1. 脚本使用 `set -e` 确保任何步骤失败时立即终止。
2. 包过滤模式 `...` 表示构建所有包，可根据需要修改为特定包名。
3. 建议将脚本纳入版本控制，确保团队使用统一的构建流程。
