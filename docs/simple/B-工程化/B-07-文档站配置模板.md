# B-S07 文档站配置模板（精简版）

> **模板 ID**: B-S07
> **模板名称**: 文档站配置模板（精简版）
> **版本**: 1.0.0
> **适用场景**: 基础文档站点配置，包含站点标题、导航和基本侧边栏

---

## 概述

本模板提供一个最简文档站点配置，适用于任何静态文档生成器。包含站点基本信息、顶部导航和基础侧边栏结构。

---

## 站点配置

```yaml
# ｛｛项目名称｝｝ 文档站配置
# 适用于文档生成器（如 VitePress / Docusaurus / MkDocs / GitBook 等）

# ====== 站点基本信息 ======
site:
  title: "｛｛站点标题｝｝"
  description: "｛｛站点描述｝｝"
  url: "｛｛站点URL｝｝"
  base: "｛｛站点根路径｝｝"

# ====== 顶部导航 ======
nav:
  - label: "首页"
    link: "/"
  - label: "快速开始"
    link: "/getting-started"
  - label: "指南"
    link: "/guide/"
  - label: "API 参考"
    link: "/api/"
  - label: "｛｛外部链接标签｝｝"
    link: "｛｛外部链接地址｝｝"

# ====== 侧边栏 ======
sidebar:
  - label: "快速开始"
    items:
      - label: "安装"
        link: "/getting-started/installation"
      - label: "基本用法"
        link: "/getting-started/basic-usage"
  - label: "指南"
    items:
      - label: "配置"
        link: "/guide/configuration"
      - label: "进阶"
        link: "/guide/advanced"
```

---

## 占位符说明

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `｛｛项目名称｝｝` | 项目名称 | `my-project` |
| `｛｛站点标题｝｝` | 文档站标题 | `My Project 文档` |
| `｛｛站点描述｝｝` | 文档站描述 | `My Project 的官方文档` |
| `｛｛站点URL｝｝` | 文档站完整 URL | `https://docs.example.com` |
| `｛｛站点根路径｝｝` | 站点根路径 | `/` |
| `｛｛外部链接标签｝｝` | 外部链接的显示文本 | `GitHub` |
| `｛｛外部链接地址｝｝` | 外部链接的 URL | `https://github.com/org/repo` |

## 目录结构

```
docs/
├── index.md                    # 首页
├── getting-started/
│   ├── installation.md         # 安装指南
│   └── basic-usage.md          # 基本用法
├── guide/
│   ├── configuration.md        # 配置说明
│   └── advanced.md             # 进阶用法
└── api/
    └── index.md                # API 参考
```

---

## 适配指南

- **VitePress**: 将上述配置映射到 `.vitepress/config.yml` 中的 `themeConfig`。
- **Docusaurus**: 将 `nav` 映射到 `docusaurus.config.js` 的 `themeConfig.navbar`，`sidebar` 映射到 `sidebars.js`。
- **MkDocs**: 将 `nav` 映射到 `mkdocs.yml` 的 `nav`，`sidebar` 合并到 `nav` 结构中。
- **GitBook**: 将配置映射到 `book.json` 的 `structure` 字段。
- **其他生成器**: 参照各生成器的配置文档进行映射。

---

## 注意事项

1. 侧边栏结构应与实际目录结构保持一致。
2. 如需多语言支持、搜索、主题定制等功能，请参考完整版模板（B-F07）。
3. 建议为每个页面添加 `title` 前置信息（front matter）。
