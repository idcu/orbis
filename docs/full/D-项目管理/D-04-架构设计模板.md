## 3. 分层依赖

### 3.1 层级定义

| 层级 | 名称 | 说明 | 允许依赖 |
|------|------|------|----------|
| L0 | 基础层 | 通用工具、基础类型 | 无 |
| L1 | 核心层 | 核心抽象、接口定义 | L0 |
| L2 | 功能层 | 通用功能实现 | L0, L1 |
| L3 | 业务层 | 业务逻辑实现 | L0, L1, L2 |
| L4 | 接入层 | 外部接口、适配器 | L0, L1, L2, L3 |
| L5 | 应用层 | 应用入口、组合编排 | L0, L1, L2, L3, L4 |
| L6 | 展示层 | 界面展示、交互逻辑 | L0, L1, L2, L3, L5 |
| L7 | 测试层 | 测试代码、测试工具 | 所有层级 |

### 3.2 依赖规则

- **单向依赖：** 高层级可以依赖低层级，低层级不能依赖高层级
- **禁止循环依赖：** 同层级模块之间不能相互依赖
- **接口隔离：** 跨层调用应通过抽象接口
