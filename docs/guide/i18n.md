# 多语言支持

VitePress 原生支持国际化（i18n），可以轻松配置多语言文档站。

## 基本配置

在 `config.ts` 中使用 `locales` 配置多语言：

```typescript
export default defineConfig({
  locales: {
    'zh-CN': {
      label: '简体中文',
      lang: 'zh-CN',
      title: '我的文档',
      description: '文档描述',
      themeConfig: {
        nav: nav('zh-CN'),
        sidebar: sidebar('zh-CN'),
      },
    },
    'en-US': {
      label: 'English',
      lang: 'en-US',
      title: 'My Docs',
      description: 'Documentation',
      themeConfig: {
        nav: nav('en-US'),
        sidebar: sidebar('en-US'),
      },
    },
  },
})
```

## 目录结构

多语言文档推荐以下目录结构：

```
docs/
├── index.md                 # 中文首页（默认语言）
├── guide/
│   ├── getting-started.md   # 中文文档
│   └── deployment.md
├── en/
│   ├── index.md             # 英文首页
│   └── guide/
│       ├── getting-started.md
│       └── deployment.md
```

## 导航函数

```typescript
function nav(lang: string) {
  if (lang === 'zh-CN') {
    return [
      { text: '指南', link: '/guide/getting-started' },
      { text: '部署', link: '/guide/deployment' },
    ]
  }
  return [
    { text: 'Guide', link: '/en/guide/getting-started' },
    { text: 'Deployment', link: '/en/guide/deployment' },
  ]
}
```

## 语言切换

VitePress 会自动在导航栏显示语言切换器（当配置了多个 `locales` 时）。
